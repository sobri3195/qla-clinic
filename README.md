# QLA Clinic Management System

Frontend-only prototype untuk **Sistem Informasi Manajemen Klinik Kecantikan** bernama **QLA Clinic**.

## Stack
- React + Vite
- Tailwind CSS
- shadcn/ui-inspired reusable components
- Framer Motion
- Lucide Icons
- React Router
- Zustand + localStorage
- Recharts

## Fitur utama demo
- Dashboard premium aesthetic clinic
- Login dummy berbasis role
- Manajemen pasien dan detail histori
- Appointment & queue management
- Consultation & medical records
- Treatment catalog dan skincare products
- Cashier, invoice, reports, staff, settings
- Persist data frontend-only dengan localStorage

## Menjalankan project
```bash
npm install
npm run dev
```

## Build production
```bash
npm run build
```

## Deploy ke Vercel
Project ini siap dideploy sebagai frontend statis di Vercel tanpa backend tambahan.
